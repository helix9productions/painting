Dealing with Files

Creating, Opening, and Saving Files

All actions pertaining to files can be dealt with within the file dropdown from the menu bar. The first function is New File which you can use to create a new image on which to draw. WARNING: this will delete any image you are currently working on. The next function is Open File which you can use to navigate to and open an image file. The final function is the Save File function which you can use yo save your current image somewhere on your computer.
